## Anatomy of a blog

### Tooling/tech
- Fully managed
	- Wordpress.com
	- Ghost.org
	- ITQ blog
- More flexible but more technical (static site generator)
	- Requires very basic hosting (static content)
	- Material for Mkdocs with blog plugin
	- Hugo
	- TiddlyWiki (single file!)
- Homegrown (tinkerers dream)
	- Go nutso, build whatever feature you need.
	- Requires runtime (and maybe database?)
	- Could require specialized hosting
